# RC-Phase-Shift-Oscillator
RC Phase Shift Oscillator using 741 Op-Amp with Multisim simulation and hardware implementation.


This project demonstrates the implementation of an RC Phase Shift Oscillator using the IC741 operational amplifier.

The oscillator generates a stable sinusoidal waveform using a feedback network consisting of resistor-capacitor (RC) stages.

## Project Overview

The circuit uses:
- IC741 Operational Amplifier
- RC feedback network
- Multisim simulation
- Hardware implementation on perf board

The RC network provides a total phase shift of 180°, while the inverting op-amp contributes another 180°, satisfying the Barkhausen criterion for sustained oscillations.

## Components Used

- IC741 Operational Amplifier
- 33kΩ Resistors
- 3.6kΩ Resistors
- 2MΩ Resistor
- 0.1µF Capacitors
- Perf Board
- Connecting Wires

## Software Used

- Multisim

## Features

- Stable sine wave generation
- Analog oscillator design
- RC phase shift implementation
- Hardware and simulation verification

- Frequency of Oscillation: 180.5 Hz
